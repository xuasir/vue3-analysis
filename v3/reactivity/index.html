# 前言

这一篇章我们将会开启对reactivity包的解析，这也是Vue中非常重要的响应式系统的实现， 得益于Vue3的架构升级，reactivity从组件的内部分离出来，独立成一个响应式的库， 现在我们阅读起源码也轻松了不少。

# 前置准备

我们希望在阅读reactivity包的具体解析之前，您能对于reactivity所暴露的方法有一些使用上的大致了解； 如果您还完全不了解，可以查看Vue 组合式 API 手册来学习。 如果你先想了解composition API的设计初衷及其细节， 可以通过组合式 API 征求意见稿来学习。

# 目标

本篇章以reactive、ref以及effect为突破口来理解Vue3响应式系统的依赖收集和派发更新如何通过proxy来处理的， 副作用函数又是如何被收集；后续再深入对于computed、watch等等API进行解析。
